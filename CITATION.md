# How to cite

If you use this software, model, or dataset in a publication, please cite our work so we receive appropriate credit. Suggested citation (plain text):

Signify — Urdu Sign Language (Hand Keypoints). Assadullah and Farhana Suleman, Lasbela University of Agriculture, Water and Marine Sciences (2025). https://github.com/AssadRaza10/signify

BibTeX entry (copy/paste):

```bibtex
@misc{signify2025,
  title = {Signify — Urdu Sign Language (Hand Keypoints)},
  author = {Assadullah and Farhana Suleman},
  year = {2025},
  howpublished = {\url{https://github.com/AssadRaza10/signify}},
  note = {Lasbela University of Agriculture, Water and Marine Sciences}
}
```

Short citation example (APA-like):

Assadullah & Suleman (2025). Signify — Urdu Sign Language (Hand Keypoints). Lasbela University of Agriculture, Water and Marine Sciences. https://github.com/AssadRaza10/signify

## Dataset Citation

This project uses the **UAlpha40** dataset. If you use this project, please also cite:

**Sajid, Shahood; Javaid, Sameena; Baloch, Yusra (2025).** "UAlpha40: A Comprehensive Dataset of Urdu alphabets for Pakistan Sign Language". Mendeley Data, V3. doi: 10.17632/3pvnnckxyb.3

BibTeX entry for the dataset:

```bibtex
@dataset{uAlpha40_2025,
  title = {UAlpha40: A Comprehensive Dataset of Urdu alphabets for Pakistan Sign Language},
  author = {Sajid, Shahood and Javaid, Sameena and Baloch, Yusra},
  year = {2025},
  publisher = {Mendeley Data},
  version = {3},
  doi = {10.17632/3pvnnckxyb.3}
}
```

Thank you for citing our work — it helps us continue research and development.
